## 💯 100 Programming Challenges Journey
This repository is part of my journey to **learn, practice, and evolve as a developer** by solving 100 progressively challenging programming questions and exercises.

The goal is simple: **build consistency, strengthen fundamentals, and reach an advanced level through practice.**

## 📚 Structure
The challenges are divided into **4 groups**, each containing **25 exercises**, organized by difficulty:

### 🟢 Introduction (1–25)
- Basic logic
- Understanding syntax
- Simple problem solving
- First contact with programming concepts

### 🔵 Basic (26–50)
- Control structures (if, loops)
- Functions
- Arrays and basic data structures
- Small real-world problems

### 🟡 Intermediate (51–75)
- More complex logic
- Data manipulation
- Algorithm thinking
- Problem decomposition

### 🔴 Advanced (76–100)
- Advanced problem solving
- Optimization
- Complex algorithms
- Real-world inspired challenges

## ⚙️ Rules
- ❌ No theoretical questions (e.g. "When was X created?")
- ✅ Only **practical exercises**
- 📈 Difficulty increases progressively
- 🧠 Focus on **thinking, not memorizing**


## 🚀 Objective
By the end of this repository, I aim to:

- Improve problem-solving skills
- Think like a developer
- Write cleaner and more efficient code
- Gain confidence tackling real-world challenges

## 📂 Organization
Each challenge will be stored in its respective folder and each exercise may include:

- Problem description
- Solution
- Notes or alternative approaches

## 🛠️ Technologies
This repository may include solutions using:

- Any programming language I choose to practice
- Focus is on logic, not language-specific features
